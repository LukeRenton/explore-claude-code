# assets/ - 정적 리소스

이 디렉토리는 스킬이 참조할 수 있는 정적 리소스를 보관합니다. Claude가 읽거나 채워 넣지만 실행하지는 않는 것들입니다.

## 여기에 넣을 것

- **템플릿**: 문서 템플릿, 설정 파일 템플릿, Claude가 사용할 때마다 채워 넣는 보일러플레이트 구조
- **스키마**: JSON 스키마, 데이터베이스 스키마, API 응답 형태
- **데이터 파일**: 조회 테이블, 매핑 파일, 참조 데이터
- **이미지**: 다이어그램, 예시 스크린샷 (Claude는 이미지를 볼 수 있습니다)
- **예시 출력**: 예상 형식을 보여주는 샘플 파일

## 작동 방식

SKILL.md가 필요할 때 자산을 참조합니다:

"시작점으로 [assets/template.md](assets/template.md)의 템플릿을 사용하세요."

Claude는 자산을 읽고 동적 부분을 채운 다음 출력을 생성합니다. 자산은 정적입니다. 호출 간에 변경되지 않습니다.

## 사용 예시

- Claude가 실제 변경사항으로 채우는 PR 템플릿
- Claude가 검증에 사용하는 설정 스키마
- Claude가 요청에 따라 커스터마이징하는 컴포넌트 보일러플레이트
